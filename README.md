# Framik

## Requirements

**Main server machine**

- bun `curl -fsSL https://bun.sh/install | bash`

**Daemon machine**

- bun `curl -fsSL https://bun.sh/install | bash`
- gpio `sudo apt install gpiod libgpiod-dev`

## Useful tools
- API call can be done using [Yaak](https://yaak.app/) by loading the yaak folder as a workspace
